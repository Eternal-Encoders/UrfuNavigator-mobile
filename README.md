# urfu_navigator_mobile

## Installation

* Install flutter sdk
* ``` flutter pub get ```
* ``` flutter run ```
### For Android
* Create env file **(not .env)** inside ./android/app/src/main/assets
* Get Yandex Map Api Key on https://developer.tech.yandex.ru/
* Inside env file paste ``` MAPKIT_API_KEY="{Your api key}" ```
